# Calculadora TMDb - R$ 0,45/min

Site estático para GitHub Pages que:

- pesquisa filmes e séries na TMDb
- pega a minutagem automaticamente
- calcula o valor por minuto
- em séries, soma os episódios por temporada e também mostra o total da série

## Como usar

1. Crie um repositório no GitHub.
2. Envie os arquivos `index.html`, `styles.css` e `app.js`.
3. Ative o GitHub Pages nas configurações do repositório.
4. Abra o site publicado.
5. Cole sua API key da TMDb no campo do topo e clique em **Salvar configurações**.

## Observação

A chave fica visível no navegador porque este é um site estático. Para um projeto simples em GitHub Pages isso é comum, mas não é a abordagem ideal para chaves que precisem ficar secretas.

## Créditos obrigatórios da TMDb

Inclua sempre a atribuição da TMDb no seu projeto.
